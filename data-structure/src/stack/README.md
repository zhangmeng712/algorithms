栈
==========

## 定义
栈 是一个后进先出（Last in first out,LIFO）的线性表，它要求只在表尾进行删除和插入操作。

<img src="http://upload.wikimedia.org/wikipedia/commons/thumb/2/29/Data_stack.svg/200px-Data_stack.svg.png" />

## API

- 入栈 push
- 出栈 pop
- 返回栈顶元素 peek
- 清空栈clear

## 实现
- [stack.js](stack.js)

## 应用

- 迷宫问题 [maze.html](example/maze.html)
- 模拟递归
